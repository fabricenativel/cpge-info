{\bf Question 2} \\
Dans tous les cas $t.(i) < n$
\begin{itemize}
\item Pour $n=1$, 0 apparaît au plus une fois dans $t$, dans les deux cas ($t.(0)=0$ ou $t.(0)=1$), le tableau obtenu n'est pas autoréférent.
\item Pour $n=2$, 0 apparaît au plus deux fois dans $t$. Si $t.(0)=0$ alors $t.(1)=0$ ou (exclusif) $t.(2)=0$. 
\end{itemize}
